# Tendo Sample Problem Solution


## Requirements

- Python3
- SqlLite

## Project Setup

- `cd to <repo_folder>`
- `python3 -m venv venv`
- `source venv/bin/activate`
- `pip install -r requirements.txt`
- `python setup.py install`

## Running Tests
- `pytest tests/`

## Import Data
- `python import_summary.py input_data.json`

## Collect Patient Survey
- `python patient_survey.py`
